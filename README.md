# FitVerse

A comprehensive fitness analytics platform with AI-powered nutrition tracking, workout visualization, and personalized health insights.

## Features

- **Dashboard Analytics** — Volume trends, workout distribution, key metrics
- **Exercise Tracking** — Personal records, 1RM estimates, performance trends
- **AI Nutrition Tracker** — Snap a photo of your food and get instant macro breakdowns via Gemini AI
- **Set-by-Set Feedback** — Real-time performance analysis with fatigue-aware rep ranges
- **Muscle Heatmaps** — Interactive body maps showing training volume distribution
- **JWT Authentication** — Secure login with Google OAuth and email/password
- **MongoDB Integration** — Cloud-synced user profiles, food logs, and macro targets
- **Theme Modes** — Light, Medium Dark, Midnight Dark, and Pure Black themes

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS, Recharts
- **Backend:** Node.js, Express, MongoDB Atlas, JWT Auth
- **AI:** Google Gemini API for food analysis and workout insights

## Getting Started

```bash
git clone https://github.com/ubothe14/FitVerse-App.git
cd FitVerse-App
npm install
npm run dev
```
